# Weather

Query real-time weather by city.

## Features

- **Brief mode** (default): weather, temperature, wind
- **Detailed mode**: adds humidity, UV, AQI, sunrise/sunset
- **Multi-day forecast**: 3-day or 7-day (weekly)

## Usage

| Trigger | Example |
|---------|---------|
| City weather | `上海天气` |
| Temperature | `今天热不热` |
| Rain check | `明天会下雨吗` |
| Travel | `周末天气适合出游吗` |
| Detailed | `深圳一周详细天气` |

**Detail keywords**: `详细` / `detailed` — enables detailed output.  
**Day keywords**: `3天` / `3 days`, `一周` / `7天` / `week`.

## Install

```bash
npx skills add coding327/lc-skills --path skills/weather
```
