# Fullstack

7-step solo full-stack development methodology for AI coding agents. Proven on 2 shipped products (47K LoC).

## Features

- **Step-by-step workflow**: from PRD to production release
- **Technical document template**: the critical bridge between requirements and code
- **Parallel development patterns**: sub-agents + git worktree
- **Bug fixing workflow**: multi-session parallel fixing with concrete evidence
- **Deployment guide**: SSH auto-deploy + distribution platform submission
- **Common pitfalls**: context explosion, design fidelity gap, model choice, logging

## Usage

| Trigger | Example |
|---------|---------|
| Full-stack development | `我想从零开发一个全栈项目` |
| Solo development | `用 AI 独立开发一个产品` |
| Build and ship | `help me build and ship a full-stack project` |
| Methodology guidance | `按照 7 步方法论开发项目` |
| Deployment help | `帮我部署上线` |
